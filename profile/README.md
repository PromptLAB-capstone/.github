# 프로젝트 개요

## 서비스명
- **창업 통합 지원 웹 서비스**: **"PREP"**

### 구현 플랫폼
- **백엔드**: Java, Spring Boot, Python, FastAPI
- **프론트엔드**:  
- **데이터베이스**: PostgreSQL  
- **서버**: AWS EC2  
- **기타**: Git, IntelliJ, Docker

---

## 아이디어 소개

"PREP"은 특허 API를 활용하여 아이디어 검진, 지원금 매칭, 발표 지원을 해주는 창업 통합 지원 서비스입니다.

---

## 핵심 기능 (MVP)

- **아이디어 검진 AI** - 특허 API를 활용하여 해당 아이디어를 검진합니다.
- **지원금 정보 매칭** - 직접 개발한 모델을 활용하여 사용자 아이디어 맞춤형 지원금 정보를 매칭합니다.
- **발표 지원** - LLM을 활용하여 발표 스크립트, Q&A 생성 및 피드백 등을 제공합니다.

---

## 세부 기능

1. **메인페이지**
    - 핵심 기능 3가지(아이디어 검진 / 지원금 매칭 / 발표 지원) 바로가기를 제공합니다.
    - 이번 달 활동 요약 정보를 제공합니다. (검진 횟수, 평균 PMF 점수, 매칭된 지원금 수)
2. **아이디어 검진 페이지**
    - 간편 검진 / 정밀 검진 탭을 선택할 수 있습니다.
    - 설문 입력 폼을 제공합니다. (필수 5항목 + 선택 4항목)
    - 검진 결과 화면을 제공합니다. (신호등 판정, 리포트 뷰어)
    - 리포트를 PDF로 저장할 수 있습니다.
3. **지원금 매칭 페이지**
    - 설문 입력 폼을 제공합니다.
    - 추천 지원금 목록을 카드 형태로 확인할 수 있습니다.
    - 지원금 상세 정보 페이지를 제공합니다. (핵심 평가 포인트 포함)
4. **발표 지원 페이지**
    - 발표 분량을 선택하여 스크립트를 생성할 수 있습니다. (3분 / 5분 / 10분)
    - 스크립트를 직접 입력하면 피드백을 받을 수 있습니다.
    - 페르소나를 선택하여 예상 Q&A를 받을 수 있습니다. (VC / 정부 / 액셀러레이터)
5. **프로젝트 관리 페이지**
    - 아이디어별로 프로젝트를 묶어 관리할 수 있습니다.
    - 검진 이력, 발표 스크립트, 지원금 매칭 결과를 프로젝트 단위로 통합 조회할 수 있습니다.
    - 프로젝트를 생성, 수정, 삭제할 수 있습니다.
6. **마이페이지**
    - 사용자 정보를 수정할 수 있습니다.
    - 이용 내역을 조회할 수 있습니다. (검진 횟수, 저장된 리포트 목록)

## Git 컨벤션 규칙

### Commit 규칙

|   Gitmoji    |    Tag     | Description |
|:------------:|:----------:| --- |
|      ✨       |   `feat`   | 새로운 기능 추가 |
|      🐛      |   `fix`    | 버그 수정 |
|      📝      |   `docs`   | 문서 추가, 수정, 삭제 |
|      ✅       |   `test`   | 테스트 코드 추가, 수정, 삭제 |
|      💄      |  `style`   | 코드 형식 변경 |
|      ♻️      | `refactor` | 코드 리팩토링 |
|      ⚡️      |   `perf`   | 성능 개선 |
|      💚      |    `ci`    | CI 관련 설정 수정 |
|      🚀      |  `chore`   | 기타 변경사항 |

#### Commit Message Format
- **헤더(Header)**: `<타입>(스코프): <주제>`
- **본문(Body)**: 커밋의 상세 내용 (선택적)
- **바닥글(Footer)**: 관련 이슈 번호

##### 예시
```plaintext
feat (ArticleController): Article 전체 조회 api 추가

전체 조회를 위한 API를 작성하였습니다.

resolve: #21
```
